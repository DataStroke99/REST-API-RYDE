    API Endpoint

         http://localhost:3000/

The RYDE API provides programmatic access to User Model data stored in a persistence database.

The API provides CRUD operations for clients and is build using Express.js for the backend and MongoDB for the NOSQL Database. For testing Mocha and Chai javascript libraries are used for comprehensive unit testing of the API. This entire project was built in a MacOS machine. The testing was done using Insomnia.

- Download the Github repo
- Open the termial and navigate to the downloaded folder
- Run npm install while in the folder with contains package.json to install the node modules
- Run command npm start-dev to start the server, now the server is running
- Open another terminal and use any of the curl commands provided to use the API

QUERY PARAMETERS

Field Type Description
id Number The id for the User data is auto incremental
name String Enter name of the User
dob String The Date Of Birth for the User
address String Address for the User
description String A custom description for the user
createdAt Date Timestamp created when entry is recorded

Testing

For testing Mocha+Chai JS frameworks are used. To personally run test in your machine stop the server and type npm run test. Test can be customized in the REST-API-RYDE/src/test/unit_test.js
